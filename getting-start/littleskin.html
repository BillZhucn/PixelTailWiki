# 如何正确配置外置登录

注：

此教程将会引导你使用外置登录,进入Pixeltail服务器

# 1.注册&登录Little Skin

注意

Yggdrasil API 认证服务器地址 可能因为您所在的地区而异

在“快速配置启动器”框中，可以看到“本站的 Yggdrasil API 认证服务器地址：https://mcskin.littleservice.cn/api/yggdrasil ”

注意

玩家的ID必须符合正版Minecraft取名规范（即只允许包含26个大小写英文字母和下划线“_”）
